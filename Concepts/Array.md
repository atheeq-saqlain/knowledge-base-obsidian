## Definition

A finite, **ordered sequence** of elements where each position is identified by a numeric **index**.

## Description

Arrays store values in a fixed order from first to last. The first position is index `0` in most languages; the last is index `length - 1`.

| Operation (typical) | Time |
|---------------------|------|
| Read or write by index | O(1) |
| Scan all elements | O(n) |
| Search unsorted | O(n) |

**Example:** `[1, 2, 3, 3]` has four elements; the value at index `3` is `3`.

Appears wherever data is given as a list, vector, or sequence — not only in code (e.g. ordered measurements, ranked items, time series).
